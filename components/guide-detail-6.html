Hướng dẫn lấy ID Profile, Fanpage, Group
Bước 1: Vào page lấy ID ở mục tài khoản
Bước 2: Ở mục lấy ID Profile, Fanpage, Group tiến hành nhập link facebook cần lấy ID và bấm vào ô tìm kiếm
Bước 3: Ở bảng này sẽ hiện ra ô có ID của bạn, sau đó tiến hành sao chép ID
Lưu ý: Nếu không hiện ID vui lòng thử lại hoặc liên hệ Fanpage
